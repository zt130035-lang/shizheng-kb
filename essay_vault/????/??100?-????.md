# 申论100题方法增补

本页只整理方法、批改检查点和答题结构，不复制题目材料、答案全文或解析原文。来源和页码记录见[[来源说明/申论100题方法来源]]。

## 一、通用做题与批改流程

1. **先审题**：圈出任务动词、作答对象、材料范围、身份、文种和字数等限制。
2. **再读材料**：按段落标记问题、原因、影响、意义、对策、做法和案例，不把不同要素混成一类。
3. **锁定要点来源**：直接摘抄材料中的规范表述；归纳概括数据、事例和口语化描述；把材料中的对策或建议反向转成问题、原因或启示。
4. **整合答案**：优先使用“总括词 + 具体内容”，按材料逻辑或题目要求分类，合并同义重复。
5. **回看限制**：检查是否答全对象、主体、范围和结果，是否超字数，是否有材料外扩写。

### 批改时的证据链

每个评分判断都尽量形成：`题目任务 → 材料依据 → 考生答案 → 得分状态 → 修改动作`。只看到答案、没有题干和材料时，只能评价结构与表达，不能断言漏点或错点。

## 二、单一题

### 问题类

- 题干信号：问题、不足、困境、瓶颈、挑战、难点、缺失等。
- 重点找负面信息。常见标志词包括“不科学、不合理、不到位、不均衡、不完善、不健全、不足、缺乏、缺少、少、低、差、弱、陈旧、单一”等。
- 数据本身通常不是答案，要把数量、速度、占比等数据转化为“规模不足、增长缓慢、覆盖不全”等问题。
- 负面案例要从“处境、经历、做法”中概括问题；材料中的对策建议可以反推对应问题。
- 批改检查：是否写成现象复述，是否遗漏问题主体和具体表现，是否把原因或对策混进问题答案。

### 影响类

- 先明确影响对象和影响方向，再按主体或维度归类。
- 区分直接影响与深层影响，避免把同一结果换词重复。
- 影响答案要写清“对谁造成什么影响”，不能只写“有利于发展”“造成不良后果”。

### 原因类

- 围绕“为什么出现该现象”找材料依据，优先区分主体、制度机制、资源能力、观念和环境条件。
- 由事例概括原因时，要从做法和处境上升为规范原因；不能把材料结果再次当成原因。

### 对策类

- 对策必须对应材料问题，优先从材料中的做法、建议和成功经验中提炼。
- 一条可执行对策至少写清“主体 + 动作”；重点对策尽量补足“对象 + 路径 + 结果”。
- 将“加强、完善、提升、推进”后的空泛方向改成具体机制、流程、载体、资源或责任安排。
- 批改顺序：问题是否覆盖 → 对策是否一一对应 → 是否有主体和动作 → 路径是否可执行 → 是否出现材料外拔高。

### 不常见要素与多要素题

- 先识别题目究竟要求做法、成效、意义、启示、特点还是问题，不用“万能概括”替代审题。
- 多要素题按题干分项作答；每项内部使用“总括词 + 具体信息”，并保持同一层级。

## 三、综合题

### 词句解释

推荐结构：**总体释义 → 展开解释 → 补充对策或总结**。

- 总体释义优先取词句上下文或权威主体的直接解释；没有完整释义时，提炼事例中的关键词概括。
- 展开解释按词句性质选择要素：负面词句重点找问题、危害、原因；正面词句重点找做法、意义、成效。
- 材料有对策时可作为收束，但对策分量通常低于总体释义和展开解释，字数不足时不能本末倒置。
- 批改时检查是否只解释字面、只复述案例，是否缺少题干对象，是否把对策写成主要答案。

### 观点或现象分析

推荐结构：**明确判断 → 材料依据 → 原因/机制/影响 → 结论或方向**。案例后必须补一句“这说明……”，让事实与观点发生连接；结论要回扣题目对象和材料条件。

### 对比分析

先找比较对象和共同维度，再分别写相同点、差异点及其原因或启示。按同一维度对照，不能先完整写完一方再零散补另一方。

## 四、应用文与公文

先服务于身份、对象和沟通目的，再检查格式。应用文不是大作文的缩写版。

### 方案类

- 适用：指导意见、建议、活动方案等，重点是规划和具体做法。
- 基本框架：标题 → 主送机关或称谓 → 开头（背景、目的、意义）→ 主体（分条对策，重点）→ 结尾（强调重要性、提出要求，字数不足可省略）→ 落款。
- 批改时先检查方案是否可执行，再检查标题、称谓、语气和落款；不能只有意义和口号。

### 其他常见文种

- 宣传、倡议：突出受众、倡议事项和行动号召。
- 总结、汇报：突出事实、问题、做法、成效和下一步安排。
- 评论：先亮明观点，再用材料分析原因、影响和判断，结尾回扣立场。
- 提案、建议：正文通常包括案由和建议；建议部分要对准问题并写出落实路径。

## 五、文章写作的补充检查

- 审题产物必须包括主题对象、核心矛盾和中心论点；中心论点不能只重复题目关键词。
- 分论点选择递进、并列或问题解决结构，以题目和材料逻辑为准，不机械套模板。
- 每个分论点尽量形成“判断 → 解释 → 依据 → 行动 → 回扣”的闭环；案例不能替代分析。
- 结尾至少完成总结、回扣主题、提出行动方向中的两项，避免重复开头或突然增加新分论点。
