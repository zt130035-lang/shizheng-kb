# GitHub开放资料来源

本页记录用于改进知识库结构的公开仓库。来源记录不参与申论向量检索，实际发布内容必须重新整理、人工审核并注明来源。

## 已采用的结构参考

### KeWang0622/kaogong-skill

- 地址：https://github.com/KeWang0622/kaogong-skill
- 许可证：MIT License。
- 内容：覆盖行测、申论和面试的考公辅导规则，包含概括归纳、综合分析、提出对策、应用文和大作文的基础评分框架。
- 使用方式：借鉴小题分类、基础评分维度和训练流程；已按本知识库的“材料优先、原文定位、可执行修改”规则重新整理为小题标准。
- 限制：不将其泛化示例或考试分数说明视为真实题目的材料要点或官方评分依据。

### KaguraNanaga/official-document-writing-skill

- 地址：https://github.com/KaguraNanaga/official-document-writing-skill
- 许可证：MIT License。
- 内容：GB/T 9704-2012 党政机关公文格式、常用模板和质量检查清单。
- 使用方式：只用于申论应用文的文种、语体和必要格式要素校验。
- 限制：正式发文的版头、字号、页码和印制规范不适用于申论答题纸，不得据此扣分。

### KaguraNanaga/kaogong-study-tracker

- 地址：https://github.com/KaguraNanaga/kaogong-study-tracker
- 许可证：MIT License。
- 内容：考公错题归档、错误原因分类、二刷提醒和学习统计。
- 使用方式：借鉴“错因分类—复练状态—二刷验证”的训练闭环，整理为 [[批改风格/小题错因跟踪]]。
- 限制：该仓库不提供按材料评分的小题规则或参考答案，不能替代当前题型评分标准。

### kuriv/civil-service-exam

- 地址：https://github.com/kuriv/civil-service-exam
- 许可证：MIT License
- 内容：公务员考试知识思维导图，包含申论、行测和面试的分类框架。
- 使用方式：只借鉴知识分类和导航思路，当前 Vault 中的规则和表达均按本项目需求重新编写。
- 限制：仓库已归档，内容较早；申论部分主要是思维导图，不能作为当前政策、时政事实或小题评分依据。
