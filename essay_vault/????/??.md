# 申论评分总则

## 基本原则

1. 先看题目要求，再看答案内容，不能脱离题目单独评价。
2. 先判断得分点是否覆盖，再评价结构和语言。
3. 批改必须指出原文位置和具体问题。
4. 修改建议必须能够直接执行，不能只写“加强逻辑”“语言不够丰富”。
5. 对材料题，答案中的关键判断应能在材料或题目要求中找到依据。

## 通用维度

| 维度 | 关注内容 | 常见问题 |
| --- | --- | --- |
| 审题 | 是否回应题目任务和限制 | 偏题、答非所问 |
| 内容 | 得分点、材料信息、观点 | 漏点、错点、重复 |
| 结构 | 层次、顺序、段落功能 | 混乱、跳跃、头重脚轻 |
| 论证 | 原因、影响、对策、论据 | 空泛、缺少因果 |
| 语言 | 规范、准确、简洁 | 口语化、套话、病句 |
| 规范 | 字数、格式、卷面、标点 | 超字数、格式错误 |

## 输出要求

批改结果至少包括：

- 总体评价
- 分项评分
- 得分点匹配
- 具体问题
- 原文修改示例
- 后续训练建议
