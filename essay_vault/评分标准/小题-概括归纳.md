# 概括归纳题评分标准

## 主要得分点

- 对象准确：回答的是题目要求的主体或问题。
- 内容完整：覆盖材料中的主要信息点。
- 表述准确：使用概括后的规范语言，不照抄无关细节。
- 分类清晰：不同层面的要点不混在一起。
- 形式规范：符合字数和作答要求。

## 常见失分

- 只摘抄材料，没有完成概括。
- 只写现象，没有写本质或影响。
- 多个要点表达重复。
- 把原因、问题、对策混为一谈。
- 遗漏材料中的主体、对象或关键限制条件。

## 批改输出

每个要点都应标记：

| 字段 | 说明 |
| --- | --- |
| 标准要点 | 参考答案中的核心信息 |
| 用户答案 | 是否出现及出现位置 |
| 匹配程度 | 完整、部分、未覆盖 |
| 修改建议 | 如何补充或压缩 |
