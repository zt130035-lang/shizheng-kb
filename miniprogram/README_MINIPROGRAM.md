# 考公时政知识库微信小程序前端

这是“方案一”的微信原生小程序前端：保留现有 Flask 后端，新增小程序可视化前端，通过 `/api/*` 接口读取新闻、日报、专题、题库、错题和学习统计。

## 导入方式

1. 启动后端：

```bash
cd D:\下载\HTML\shizheng_kb
python server.py
```

2. 微信开发者工具导入目录：

```text
D:\下载\HTML\shizheng_kb\miniprogram
```

3. 开发调试：

- 默认接口地址：`http://127.0.0.1:5678`
- 微信开发者工具中勾选：不校验合法域名、TLS 版本及 HTTPS 证书。
- 如果后端部署到云端，在“我的 -> 接口设置”里改成 HTTPS 地址。

## 已实现

- 首页：统计卡片、近 7 天训练柱状图、最新时政、学习入口。
- 时政：新闻归档、搜索、详情、收藏。
- 学习材料：每日日报、时政专题、晨读卡。
- 问答：调用 `/api/query` 做知识库语义检索。
- 题库：PDF 列表、上传、选择题练习、错题自动记录。
- 我的：收藏、错题、统计、后端地址设置。

## 上线前建议

- 后端部署 HTTPS 域名，并添加到微信小程序合法域名。
- 增加微信登录接口，按 openid 隔离收藏、错题和训练记录。
